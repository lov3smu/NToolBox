说明

Cron 表达式是一种用于指定定时任务执行时间的字符串表达式。它由 6-7 个字段组成，分别表示秒、分钟、小时、天数、月份、星期几和年份（可选）。

┌──────────── [可选] 秒 (0 - 59)
│ ┌────────── 分钟 (0 - 59)
│ │ ┌──────── 小时 (0 - 23)
│ │ │ ┌────── 天数 (1 - 31)
│ │ │ │ ┌──── 月份 (1 - 12)
│ │ │ │ │ ┌── 星期 (0 - 6, 星期天=0)
│ │ │ │ │ │
* * * * * * *  命令

特殊字符说明：

  • *：表示匹配任意值，如分钟字段的 * 表示每分钟都执行
  • ,：用于分隔多个值，如 1,3,5 表示第1、3、5点执行
  • -：用于指定范围，如 10-20 表示从10到20
  • /：用于指定间隔，如 */5 表示每5分钟执行一次
  • ?：仅用于日和周字段，表示不指定具体值（避免冲突）
  • L：表示最后，如 L 表示每月最后一天
  • W：表示最近工作日，如 15W 表示每月15日最近的工作日
  • #：用于指定第几个星期几，如 2#3 表示每月第3个星期一

常用示例：

  • 0 0 12 * * ?：每天中午12点执行
  • 0 */5 * * * ?：每5分钟执行一次
  • 0 0 8-18 ? * MON-FRI：周一至周五8点到18点每小时执行
  • 0 0 0 L * ?：每月最后一天执行
  • 0 0 0 ? * 2#1：每月第一个星期一执行
