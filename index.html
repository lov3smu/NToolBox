Cron 表达式说明

Cron 表达式是一种用于指定定时任务执行时间的字符串表达式。它由 6-7 个字段组成，分别表示秒、分钟、小时、天数、月份、星期几和年（可选）。

            ┌──────────── [可选] 秒 (0 - 59)
            │ ┌────────── 分钟 (0 - 59)
            │ │ ┌──────── 小时 (0 - 23)
            │ │ │ ┌────── 天数 (1 - 31)
            │ │ │ │ ┌──── 月份 (1 - 12) OR jan,feb,mar,apr ...
            │ │ │ │ │ ┌── 星期几 (1 - 7, 1=星期日) OR sun,mon ...
            │ │ │ │ │ │
            * * * * * * ? 命令

特殊字符说明：

  • *：表示匹配任意值。例如，* 在分钟字段中表示每分钟都执行。
  • ,：用于分隔多个值。例如，1,3,5 在小时字段中表示 1 点、3 点和 5 点执行。
  • /：用于指定间隔值。例如，*/5 在分钟字段中表示每 5 分钟执行一次。
  • -：用于指定范围。例如，10-20 在日期字段中表示从 10 号到 20 号。
  • ?：仅用于日期和星期几字段，表示不指定具体值。通常用于避免冲突。
  • L：表示最后。例如，L 在日期字段中表示月末最后一天。
  • W：表示最近的工作日。例如，15W 表示每月 15 号最近的工作日。
  • #：用于指定第几个星期几。例如，2#3 表示每月第 3 个星期一。

常用示例：

  • 0 0 12 * * ?：每天中午 12 点执行
  • 0 0/5 * * * ?：每 5 分钟执行一次
  • 0 0 8-18 ? * MON-FRI：周一至周五的 8 点到 18 点之间每小时执行一次
  • 0 0 0 1 * ?：每月 1 号 0 点执行
  • 0 0 0 ? * 1：每周日 0 点执行
  • 0 0 0 L * ?：每月最后一天 0 点执行
  • 0 0 0 ? * 2#1：每月第 1 个星期一 0 点执行
